# Simple Python Task Manager

A beginner-friendly command-line task manager app written in Python.  
You can add tasks, view all tasks, mark them as done, and exit â€” all from your terminal.

## Features

- Add new tasks  
- View your tasks and see which are done  
- Mark tasks as done  
- Exit when finished

## How to Run

1. Make sure you have *Python 3* installed.
2. Copy the code from task_manager.py (main script).
3. Open your terminal and run:

    
    python task_manager.py
    

4. Follow the prompts to manage your tasks interactively.
